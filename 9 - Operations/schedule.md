## Typical day in the life

```
03:00 — Dream cycle (nightly maintenance)
06:00 — Sync upstream (Mondays only)
07:00 — Briefing (morning digest)
08:00 — LinkedIn auto-post (Mon–Fri only)
09:00 — Weekly review (Mondays only)
09:00, 12:00, 15:00, 18:00, 21:00 — Inbox processing (every 3h)
21:00 — LinkedIn trend pull + post generation (Sundays only)
00:00, 03:00 — Inbox processing (overnight)
04:00 — Maintain (Sundays only, weekly health check)
```

## How this file is kept in sync

Regenerated by Claude after:
- A workflow is added, archived, or has its schedule changed
- A scheduled run completes (to refresh current cadence)
- Manual request via `/brain-status`

It is NOT auto-generated on every run — just periodically and on change.
